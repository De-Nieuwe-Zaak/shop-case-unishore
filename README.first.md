### Getting Started

Make sure you have Node.js 21 installed (or use `nvm use`), then run:

```bash
npm install
npm run dev
```

  
When you're done, please zip the whole project and send it to [pieter@denieuwezaak.nl](http://pieter.po.nl)  
  
Good Luck! 🚀
